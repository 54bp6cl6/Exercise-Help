# Exercise Help 重訓幫手
重訓幫手是一款幫助紀錄健身成果的Line bot，

提供歷史最高紀錄以及破紀錄的提示，

結算時也可以總覽今天的訓練成果，

簡單的操作適合運動時不想花心力操作App的人。
